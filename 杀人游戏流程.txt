杀人游戏流程：
	第一天从晚上开始
		宣布{天数}晚上开始
		如果 有"医生" : 请求医生行动
			给出当前玩家列表、空针计数、总针数（/剩余针数）
			保存被扎玩家
		如果 有"杀手" : 请求杀手行动
			给出当前玩家列表
			保存被杀玩家
		如果 有"狙击手" : 请求狙击手行动
			给出当前玩家列表、总子弹数（/剩余子弹数）
			保存被狙玩家
		如果 有"警察" : 请求警察行动
			给出当前玩家列表
			保存被查玩家
			返回被查玩家身份
		处理杀人状态
			所有存活玩家置为0
			如果 被扎针 : -1
			如果 被杀 : +1
			如果 被狙 : +1
			如果 >0 : 死亡，存入死亡名单
			如果 <0 : 空针计数+1
			如果 空针计数 >1 : 死亡，存入死亡名单
		宣布{天数}夜晚结束
	每一轮开始：
		宣布{天数}白天开始	#例子：第三天白天开始
		检查死亡名单
			如果 不为空 : 公布名单
				如果 满足胜利条件 : 宣布游戏结束
				否则 : 请求遗言
			否则 : 返回平安夜
		宣布讨论和投票开始并开始倒计时
		计票：格式为{玩家序号}:{玩家序号}
		倒计时结束输出投票结果
			如果 有且只有一人得票最高 : 死亡
			否则 : 请求输入死者序号
		如果 判断胜利条件 : 宣布游戏结束
		否则 : 宣布白天结束，进入夜晚。
